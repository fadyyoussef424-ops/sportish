# The Boys in the Boat Landing Page

This is a responsive Next.js landing page created as the final digital moment
for a sport presentation about _The Boys in the Boat_.

The page is organized around three main sections:

- **Home** introduces the event as organized via ACC and sets the mood with a
  dramatic rowing finish image.
- **About Team** presents the Sportish team identity and connects the team,
  event, and presentation together.
- **Story** retells the provided brief description of _The Boys in the Boat_
  using the book cover and historical story images from the project assets.

The design is built to feel like a strong closing slide for an in-person
presentation while still working well for guests opening it on phones. It uses
large visual moments, readable story text, ACC branding, the team logo, and a
gallery of key events from the story.

The main idea of the app is simple: show how rowing becomes a story about trust,
teamwork, rhythm, and holding together under pressure.
